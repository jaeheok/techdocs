

최초 생성시엔 DB 데이터는 처음크기 3메가 자동증가량은 1MB단위
로 무제한 증가이며
로그는 1메가 기본에 10퍼센트 단위로 무제한 증가입니다.

일단 요청 하신 데이터베이스 파일 증가율의 변경은 다음과 같이
수행 하시면 됩니다.

데이터파일 증가율 변경 쿼리

ALTER DATABASE Sample
MODIFY FILE
(NAME = Sample_dat, FILEGROWTH=100MB);
GO

혹은 해당 데이터베이스의 속성탭에서 이동 하셔서
파일 부분에서 데이터 파일의 증가량을 변경 하시면 될 것으로 보
입니다.(GUI방식)

위 명령 내용은 마이크로소프트에서 배포하는 SQL 2005 관리 가이
드의 공식수행 명령이며 온라인 설명서를 통해서도 원하시는 쿼리나 수행 방
법에대한 지침이 자세히 나와 있습니다.

따로 SQL2005서버가 설치 된 서버 및 설명서 다운 설치에 시간이
걸려 답변이 늦어 졌습니다.

아래는 온라인 설명서 다운로드 페이지 입니다.
http://www.microsoft.com/downloads/details.aspx?
displaylang=ko&FamilyID=BE6A2C5D-00DF-4220-B133-29C1E0B6585F

해당 온라인 설명서에서 \"증가\"를 통해 검색하시면 더 상세하고
원하시는 내용의 확인이 가능 하십니다.

추가로 일단 데이터의 백업을 진행 하신 뒤 작업을 진행 하시기바라며
최초 데이터생성 사이즈의 경우는 이미 생성 된 부분이므로 정확하
게변경 가능 여부를 이야기 드리기 어려울 것 같습니다.
